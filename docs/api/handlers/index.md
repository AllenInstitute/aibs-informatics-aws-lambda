# Handlers

The `handlers` module contains Lambda handler implementations for various AWS services and tasks.

## Overview

| Module | Description |
|--------|-------------|
| [Batch](batch/index.md) | AWS Batch job definition and execution handlers |
| [Data Sync](data-sync/index.md) | Data synchronization and file system handlers |
| [Demand](demand/index.md) | Demand execution scaffolding handlers |
| [ECR](ecr/index.md) | ECR image replication handlers |
| [Notifications](notifications/index.md) | Notification routing and delivery handlers |
